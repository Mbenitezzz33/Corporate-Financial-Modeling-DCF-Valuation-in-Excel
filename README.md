# Corporate Financial Modeling & DCF Valuation

---

## Overview

This project demonstrates the development of an integrated Excel financial model used to evaluate capital investment opportunities through historical financial statement analysis, discounted cash flow (DCF) valuation, and sensitivity analysis.

The workbook incorporates real-world corporate finance concepts, including capital budgeting, company-specific cost of capital estimation, and scenario testing to determine the impact of changing assumptions on project valuation.

This project was completed as part of advanced corporate finance coursework and serves as an example of practical financial modeling and valuation techniques commonly used by financial analysts, investment analysts, and corporate finance professionals.

## Key Concepts Applied

### Discounted Cash Flow (DCF)

The DCF method estimates the value of an investment by discounting projected future cash flows back to their present value using the firm's required rate of return.

### Weighted Average Cost of Capital (WACC)

The WACC represents the firm's blended cost of financing from debt and equity and serves as the discount rate used in valuation.

### Capital Asset Pricing Model (CAPM)

The CAPM is used to estimate the required return on equity:

```text
Re = Rf + β(Rm − Rf)
```

Where:

* **Rf** = Risk-free rate
* **β** = Equity beta
* **Rm − Rf** = Market risk premium

### Net Present Value (NPV)

The NPV measures the value created by an investment:

```text
NPV = Σ [FCF_t / (1 + r)^t] − Initial Investment
```

A positive NPV indicates value creation for shareholders.
